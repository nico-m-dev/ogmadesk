OgmaDesk

Version 0.1.0-alpha

Core Philosophy

OgmaDesk was built on a simple premise: your data is yours. In an era of cloud-first applications that require accounts, subscriptions, and constant internet connections, OgmaDesk offers a suite of essential productivity tools that run entirely on your local machine.

No telemetry, no analytics, no cloud syncing unless you explicitly use a third-party tool like Syncthing or Dropbox on your local folders.

Technology Stack

  • Tauri v2 A lightweight framework for building secure desktop apps with a Rust backend.
  • Vanilla JS Zero-build frontend for maximum performance and simplicity.
  • Tailwind CSS Utility-first css framework for instant styling.
